INSTRUCCIONES PARA VIDEOS LOCALES DE VISTA PREVIA (NETFLIX-STYLE)
================================================================

Para que los videos de vista previa se reproduzcan de forma local (lo cual es mucho más rápido, fluido y evita bloqueos de YouTube), sigue estos pasos:

1. Descarga clips cortos (de unos 10 a 15 segundos de duración preferiblemente) en formato .mp4 de los profesores de matemáticas.
2. Coloca los archivos descargados dentro de esta misma carpeta (assets/videos/).
3. Renombra los archivos .mp4 según el identificador de YouTube que tienen asignado en el proyecto. Aquí tienes la lista de archivos que debes colocar:

   - Profe Alex:
     Archivo: CJEsybfPQSE.mp4

   - Susi Profe:
     Archivo: CN4n6Tfc5WI.mp4

   - JulioProfe:
     Archivos (puedes usar el mismo video o diferentes para cada tema):

     - K3t1XN1gN7c.mp4 (Edades / Línea de Tiempo)
     - R9H7nJ7yU38.mp4 (Modelado Geométrico)

   - El Traductor de Ingeniería:
     Archivo: kYJ53u0aJ8o.mp4

   - Academia Vonex:
     Archivo: F71H4y2t5aY.mp4

¿CÓMO FUNCIONA?
El sistema intentará reproducir el archivo local (ej. assets/videos/CJEsybfPQSE.mp4) de manera automática. Si el archivo no existe en esta carpeta, el sistema volverá al modo anterior y cargará el video directamente desde YouTube de forma segura.
